Web Cloud Client Examples

Welcome to the DeviceAtlas Java Cloud Client API web examples. This Client API provides an easy way to request the DeviceAtlas Cloud web service and get device information such as screen dimensions, whether it is mobile, vendor, model, etc.

Check out the following web examples based on the Java Spring framework.

Analytics

This web example uses the API to get properties for user-agents from a given list. Some properties such as vendor, browser name and device type are aggregated and the results are displayed as graphs and numbers.

Redirection

This web example uses the API to get properties for the current request and then uses some basic property values to decide which website provides the most suitable content for the device making the request.

Content Targeting

This example uses the API to detect the device and use some of its properties to show certain advertisements and download links which may be related or of interest to the user, considering his/her device.

Content Adaptation

This web example uses the API to get properties for the device making the current request and then uses some basic property values to choose a suitable template to wrap around the content.

Visitor information

This web example displays the whole list of the detected user device properties.
